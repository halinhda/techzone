# TechZone - Hướng dẫn truy cập

## 1. Khởi động môi trường
1. Mở XAMPP và bật Apache + MySQL.
2. Đảm bảo bạn đang ở thư mục dự án: `C:\xampp\htdocs\bainhom`.
3. Tạo / import cơ sở dữ liệu `techzone_db` từ file `database.sql`.
4. Link csdl: http://localhost/phpmyadmin/

## 2. Truy cập website
- Trang chủ: http://localhost/bainhom/index.php
- Giỏ hàng: http://localhost/bainhom/cart.php
- Thanh toán: http://localhost/bainhom/checkout.php
- Hỗ trợ: http://localhost/bainhom/support.php

## 3. Tài khoản mẫu

## 4. Ghi chú
- Nếu bạn thấy ảnh sản phẩm không hiện, hãy kiểm tra thư mục `assets/images`.
- Nếu giỏ hàng không cập nhật, hãy reload lại trang sau khi thêm sản phẩm.
